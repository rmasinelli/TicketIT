# Changelog — Cinder by Ember

## v1.1.0 — Ember Brand Refresh
- Renamed from RAM Tech TicketIt to Cinder by Ember
- Applied full Ember brand: Amber Warm accent, Charcoal surfaces, warm white text
- Typography updated: Raleway (display), Inter (body), JetBrains Mono (system text)
- Added Knowledge Base with draft/publish workflow
- Added Incident Response tracking (PICERL framework)
- Refactored monolith into modular data files (constants, courses, scenarios, seeds)
- Migrated storage from window.storage to localStorage for browser compatibility

## v1.0.0 — Initial Release
- Three-course system: Networking, Hardware, Cybersecurity
- Role-based access: student, tech, admin
- 30 pre-built lab scenarios (10 per course)
- SLA timers with live countdown
- Simulated inbox notifications
- Lab Manager with week-by-week activation
- Persistent storage across sessions
